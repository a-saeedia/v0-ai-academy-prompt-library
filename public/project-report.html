۱. خلاصه اجرایی

پروژه آکادمی هوش مصنوعی تترا یک پلتفرم جامع و دوزبانه (فارسی-انگلیسی) برای مدیریت و اشتراک‌گذاری پرامپت‌های هوش مصنوعی است که با استفاده از جدیدترین تکنولوژی‌های وب توسعه یافته است.
وضعیت پروژه: آماده برای استقرار نهایی (Production Ready) ✓

۲. هویت بصری و برندینگ

پالت رنگی تترا

  • رنگ اصلی: سبز روشن (#afd700) - استفاده در دکمه‌ها، لینک‌ها و عناصر تعاملی
  • رنگ متن: مشکی (#000000) - برای خوانایی بهینه
  • پس‌زمینه: سفید (#ffffff) - برای ظاهر تمیز و حرفه‌ای
  • رنگ هاور: سبز تیره (#8fb600) - برای بازخورد بصری

لوگو و تایپوگرافی

  • لوگوی رسمی تترا: پیاده‌سازی شده با لینک به tethera.io
  • فونت انگلیسی: Outfit (وزن‌های 300, 400, 600, 800)
  • فونت فارسی: Vazirmatn (وزن‌های 300, 400, 600, 800)

۳. سیستم احراز هویت

روش‌های ورود

  • ✓ ایمیل و رمز عبور: سیستم ثبت‌نام و ورود سنتی
  • ✓ Google OAuth: ورود سریع با حساب گوگل (نیاز به فعال‌سازی در Supabase)
  • ✓ دسترسی عمومی: امکان مشاهده کتابخانه بدون ثبت‌نام

امنیت

  • Row Level Security (RLS): محافظت از داده‌های کاربران در سطح دیتابیس
  • Session Management: مدیریت نشست‌ها با Supabase
  • Protected Routes: محافظت از مسیرهای خصوصی

۴. معماری دیتابیس

جداول پایگاه داده

  • profiles: اطلاعات پروفایل کاربران (نام، زبان، تاریخ عضویت)
  • custom_prompts: پرامپت‌های شخصی کاربران
  • community_prompts: پرامپت‌های اشتراکی با جامعه
  • prompt_likes: سیستم لایک پرامپت‌های اجتماعی
-- مثال: جدول پرامپت‌های اجتماعی CREATE TABLE community_prompts ( id UUID PRIMARY KEY, user_id UUID REFERENCES profiles(id), title TEXT NOT NULL, content TEXT NOT NULL, category TEXT, likes_count INTEGER DEFAULT 0, created_at TIMESTAMP DEFAULT NOW() );
